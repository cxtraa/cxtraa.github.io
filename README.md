## The Enchiridion

The Enchiridion is a project authored by Citra Romanov. It is a dedicated library of knowledge containing comprehensive articles on a range of fields, including Mathematics, Physics, Computer Science, and Artifical Intelligence & Machine Learning. 

Currently, the project is in its very early days, and the website is only barely functional at the moment. I plan to add many articles in the coming months. Please contact me on GitHub if you would like to contribute an article.
